# OneNoteUtilitiesGraph
OneNote module for PowerShell using the Graph REST API

Following on from OneNoteUtilities this module uses the Microsoft Graph API instead of the desktop COM objects.

This module provides the following functions:

Get-ONDefaultSection
Get-ONPage
Get-ONPageXML
Get-ONNoteBook
Get-ONNoteBooks
Get-ONSectionGroup
Get-ONSectionGroups
Get-ONSection
Get-ONSections
Get-ONResource
Get-ONResourcea
New-ONElement
New-ONPage
New-ONPageXML
New-ONNoteBook
New-ONSectionGroup
New-ONSection
Invoke-ONApp
Invoke-ONWeb
