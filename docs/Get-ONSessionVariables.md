# Get-ONSessionVariables

## SYNOPSIS
Gets a set of useful variable values.

## SYNTAX

```
Get-ONSessionVariables [<CommonParameters>]
```

## DESCRIPTION
Gets some variable values. Useful in monitoring/debugging.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-ONSessionVariables
```
