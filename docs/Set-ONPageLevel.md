# Set-ONPageLevel

## SYNOPSIS
Sets the page indentation level of a OneNote Page.

## SYNTAX

```
Set-ONPageLevel [[-Id] <String>] [[-Level] <Object>] [<CommonParameters>]
```

## DESCRIPTION
Sets the page indentation level of a OneNote Page.
Not applicable to the first Page in a Section.
