# Get-AuthCode

## SYNOPSIS
Gets an Authorization code from the Microsoft Graph API

## SYNTAX

```
Get-AuthCode [<CommonParameters>]
```

## DESCRIPTION
Gets an Authorization code from the Microsoft Graph API.
If necessary, displays
a web form for the user to authorize access to OneNote
